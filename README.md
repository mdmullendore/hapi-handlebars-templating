Running Hapi.js App With Handlebars views and a ghost blog
======================================

SetUp To Start Server Once Dev Env is SetUp
---------------------

    $ node app.js


Dev Env
------------------------------


Install Hapi.js, Handlebars.js and Ghost (Ghost is for bloging, optional)

    $ npm install hapi
    $ npm install handlebars --save
    $ npm install ghost
